## 2. 우선순위 및 작업 계획

### 2.1 우선순위 분류

#### 🔴 P0: 긴급 (기능 누락)
1. **투데이 - 타입 필터 추가** - 루틴/미션/참고 필터링 기능 누락

#### 🟠 P1: 높음 (UX 차이가 큼)
2. **투데이 - 날짜 네비게이션** - 칩 버튼 스타일로 변경
3. **리포트 - 전체 레이아웃** - 구조가 많이 다름
4. **투데이 - 체크박스 스타일** - 사각형으로 통일

#### 🟡 P2: 중간 (스타일 통일)
5. **헤더 문구 통일** - 모든 화면 헤더 Web과 동일하게
6. **버튼 스타일 통일** - "+ 새로 만들기" 링크 스타일
7. **카드 그림자/테두리** - 스타일 일관성
8. **아이콘 스타일** - 이모지 vs 라인 아이콘

#### 🟢 P3: 낮음 (선택적)
9. **토글 색상** - 검정색으로 통일
10. **완료 상태 표시** - 취소선 스타일

### 2.2 작업 순서

```
Phase 1: 핵심 기능 (P0)
├── 투데이 - 타입 필터 구현
└── 예상 시간: 2-3시간

Phase 2: UX 개선 (P1)
├── 투데이 - 날짜 네비게이션 개선
├── 투데이 - 체크박스 스타일 변경
├── 리포트 - 레이아웃 재구성
└── 예상 시간: 4-5시간

Phase 3: 스타일 통일 (P2)
├── 모든 화면 헤더 문구 통일
├── 버튼/카드 스타일 통일
├── 아이콘 스타일 통일
└── 예상 시간: 3-4시간

Phase 4: 마무리 (P3)
├── 토글 색상 통일
├── 완료 상태 표시 통일
└── 예상 시간: 1-2시간
```
