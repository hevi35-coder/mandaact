## Goal

## Step 1: Persona + Situation

- Save: persist answers on submit.

## Step 2: Core Goal Confirmation
- UI: AI-proposed core goal in editable field.

- Save: store core goal in session state.

## Step 3: Sub-Goals Draft
- UI: list of 8 sub-goal cards (editable names).
- Actions: rename, reorder optional.
- Save: store 8 sub-goals in session state.
