# MandaAct AdMob 수익화 전략

**최종 업데이트**: 2025-12-07
**버전**: 1.1 (Ad-Free Time & Feedback Loop 추가)
**상태**: 전략 확정 및 구현 대기

---

## 1. 핵심 전략: "집중을 방해하지 않는 수익화"

MandaAct는 생산성 앱입니다. 따라서 **"사용자의 몰입(Flow)을 깨지 않는 것"**이 최우선 가치입니다.
이를 위해 **"집중 모드(Ad-Free Time)"** 개념을 도입하여, 광고 시청을 유저가 능동적으로 선택하는 '거래(Trade)'의 형태로 재정의합니다.

### 1.1 하이브리드 모델 (보상형 중심)
| 구분 | 전략 |
|---|---|
| **기본 (Default)** | 일부 화면 하단 배너 노출 (Today 화면 제외) |
| **보상 (Reward)** | 유저가 **자발적으로** 광고를 보면 혜택 제공 |
| **혜택 (Benefit)** | **24시간 동안 모든 배너 광고 제거** (집중 모드) |

---

## 2. 광고 배치 세부 전략

### 2.1 배너 광고 (Banner Ads)
**원칙**: 작업의 흐름이 끊기지 않는 곳에만 배치합니다. **가장 중요한 '오늘의 실천' 화면에는 배너를 노출하지 않습니다.**

| 화면 | 위치 | 노출 여부 | 비고 |
|------|------|----------|------|
| **Home** | 하단 고정 | ✅ 노출 | 대시보드 성격 |
| **Today** | - | ❌ **미노출** | **집중 영역 보호 (Clean Zone)** |
| **MandaList** | 하단 고정 | ✅ 노출 | 목록 조회 성격 |
| **Reports** | 하단 고정 | ✅ 노출 | 리포트 조회 성격 |

### 2.2 보상형 광고 (Rewarded Ads) - 핵심 엔진
유저에게 "광고를 보면 확실한 이득이 있다"는 것을 학습시킵니다.

| 기능 | 보상 내용 | 쿨다운 | UX 메시지 |
|------|-----------|--------|-----------|
| **집중 모드 (Ad-Free)** | **24시간 배너 제거** | 24시간 | "광고 1번 보고 오늘 하루 방해받지 않기" |
| **XP 부스트** | 30분간 XP 2배 | 24시간 | "부스트 켜고 빠른 레벨업!" |
| **AI 리포트** | 리포트 즉시 생성 | - | "광고 보고 기다림 없이 분석하기" |
| **스트릭 프리즈** | 스트릭 1일 보호 | 7일 | "광고로 소중한 스트릭 지키기" (🚫 **비활성화**) |
> **Note**: '스트릭 프리즈'는 현재 비활성화되었습니다. 유저 친화적 정책으로 **'어제 날짜 체크 무료 허용 (Grace Period)'**이 적용되어 있어, 별도의 유료/광고 아이템이 불필요합니다.

### 2.3 전면 광고 (Interstitial Ads)
자연스러운 전환 시점에만 제한적으로 노출합니다.
*   **Triggers**: 만다라트 생성 완료 후, 주간 리포트 확인 후
*   **빈도 제한**: 세션당 1회, 3시간 쿨다운 (Frequency Capping)

## 3. 고도화 기능 (Advanced Features)

### 3.1 "Ad-Free Time" (집중 모드) 시스템
유저가 배너 광고를 **'제거할 권리'**를 갖게 합니다.
1.  **진입점**: 설정 메뉴 또는 배너 영역 옆의 '광고 제거' 버튼
2.  **오퍼**: "오늘 하루 광고 없이 집중하고 싶으신가요? 짧은 영상 1개만 보면 24시간 동안 배너가 사라집니다."
3.  **효과**: 
    *   보상형 광고 시청 (`rewarded`)
    *   `ad_free_until` 값을 `now + 24h`로 설정
    *   앱 전역에서 배너 숨김 (`display: none`)
    *   토스트 메시지: "집중 모드 ON! 24시간 동안 광고가 보이지 않습니다."

### 3.2 광고 싫어요 피드백 루프 (Feedback Loop)
배너 광고에 대한 부정적 경험을 Premium 전환 기회로 활용합니다.
1.  **UI**: 배너 광고 우측 상단에 작은 `x` (닫기/숨기기) 버튼 배치
2.  **동작**: 유저가 `x`를 누르면 광고가 닫히는 대신 **"Premium 프로모션 모달"** 노출
3.  **메시지**: 
    *   "이 광고가 방해되시나요?"
    *   "Premium으로 업그레이드하고 영구적으로 광고를 제거하세요."
    *   [Premium 알아보기] / [괜찮습니다]

## 4. UI/UX 가이드라인

### 4.1 집중 모드 알림 (Toast)
보상형 광고 시청 완료 시, 단순한 "보상 지급됨"이 아닌 **가치**를 전달합니다.
*   **Icon**: 🛡️ (방패) 또는 🌙 (달/집중)
*   **Text**: "집중 모드 활성화! 내일 이 시간까지 광고 없이 쾌적하게 이용하세요."
*   **Visual**: 화면 테두리에 잠시 금색/파란색 글로우 효과(Glow Effect)로 '보호받는 느낌' 연출

### 4.2 배너 영역 처리
*   배너가 있을 때와 없을 때 레이아웃이 덜컹거리지 않도록 유의 (SafeArea 처리).
*   배너가 사라진 공간은 콘텐츠 영역으로 자연스럽게 확장.
