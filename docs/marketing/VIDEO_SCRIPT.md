# App Preview Video Script (v1.1.0)

This script is designed for a **15-second High-Pace App Preview** for the App Store.
**Goal**: Show the "Magic" (AI) and the "Reward" (Gamification) instantly.

## ðŸŽ¬ Pre-Recording Setup
1.  **Simulator**: iPhone 16 Pro Max.
2.  **Settings**:
    *   Set Time to **9:41** (Standard Apple Time).
    *   Hide Status Bar if possible (or crop later).
    *   Turn off "Slow Animations" (Debug mode).
3.  **User State**:
    *   Create a fresh account or reset data so "First Report Free" is visible.
    *   Have enough XP so you are 1 check away from a Level Up (for Scene 3).

## ðŸ“¹ Storyboard (15 Seconds)

### Scene 1: The AI Hook (0s - 4s)
**"Smart Planning in Seconds"**

*   **Action**:
    1.  Tap an empty cell.
    2.  Type "Run 5km".
    3.  Tap "Get AI Suggestions".
    4.  ** VISUAL CUE**: Capture the *exact moment* the `[Routine | Weekly]` tag appears automatically.
    5.  Tap "Apply".
*   **Text Overlay (Post-Production)**: "Just Type. AI Plans It."

### Scene 2: The Satisfaction (4s - 8s)
**"Satisfying Execution"**

*   **Action**:
    1.  Go to "Today" screen.
    2.  Rapidly check off 3-4 items (Tap, Tap, Tap).
    3.  Watch the top Progress Bar fill up and turn Green/Blue.
*   **Text Overlay (Post-Production)**: "Track Habits. Feel Good."

### Scene 3: The Dopamine Hit (8s - 11s)
**"Gamified Growth"**

*   **Action**:
    1.  Check off the *last* item needed for a Level Up.
    2.  **VISUAL CUE**: Capture the **Level Up Modal** or **Badge Unlock** popup appearing.
    3.  See the XP bar go up.
*   **Text Overlay (Post-Production)**: "Level Up Your Life."

### Scene 4: The Insight (11s - 15s)
**"Expert Feedback"**

*   **Action**:
    1.  Go to "Report" tab.
    2.  Tap the **Gradient "Generate" Button**.
    3.  Scroll down the generated report (emphasize the Graph/Score).
*   **Text Overlay (Post-Production)**: "Get AI Coaching. Download Now."
