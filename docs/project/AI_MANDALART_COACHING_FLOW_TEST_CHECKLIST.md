# AI Mandalart Coaching - Flow Test Checklist

Last updated: 2025-12-25

## 목적
AI 코칭 7단계 플로우의 핵심 동작/가드/저장을 TestFlight 제출 전 수동 점검한다.

---

## 0) 접근/게이트
- [ ] 홈/생성 진입 시 코칭 게이트 진입 정상
- [ ] 주간 요약: 사용 횟수/가이드 문구 정상 노출
- [ ] 무료 1회 제공: 첫 진입 CTA 동작
- [ ] 광고 게이트: 광고 시청 완료 후 세션 시작
- [ ] 광고 실패/미노출 시 에러 토스트 표시
- [ ] 주간 한도 도달 시 제한 메시지/CTA 노출
- [ ] 슬롯 가득 참: 삭제 유도 + 삭제 후 진입 가능

## 1) 세션 상태
- [ ] 진행 중 카드: 단계/요약/다음 안내 표시
- [ ] 일시중지 → 재개 동작 정상
- [ ] 요약 보기/숨기기 토글 정상

## 2) Step 1 (상황 파악)
- [ ] 페르소나 4종 선택 가능
- [ ] 필수 질문 미응답 시 에러
- [ ] 추가 질문 토글 동작
- [ ] 입력값 저장 후 다음 단계로 이동

## 3) Step 2 (핵심 목표)
- [ ] SMART 가이드/페르소나 팁 노출
- [ ] 빈 값 방지 검증
- [ ] 저장 후 단계 이동

## 4) Step 3 (서브 목표)
- [ ] 최소 4개 입력 시 진행 가능
- [ ] 추가 입력 버튼으로 6/8 확장
- [ ] 검증 메시지 정상

## 5) Step 4 (실천 항목)
- [ ] 기준(기본/최소/도전) 가이드 노출
- [ ] 서브 목표별 접기/펼치기 동작
- [ ] 추가 실천 항목 입력 가능
- [ ] 필드 미입력 검증

## 6) Step 5 (현실성 점검)
- [ ] 제안/이유 문구 노출
- [ ] 제안 적용 시 값 변경
- [ ] 제안 거부 시 안내 문구 노출

## 7) Step 6 (루틴)
- [ ] 기본 루틴(주2/주3/매일) 선택 가능
- [ ] 맞춤 루틴: 요일/주간 횟수 선택 가능
- [ ] 선택 미완료 시 검증

## 8) Step 7 (최종 검토)
- [ ] 코어 목표/서브 목표/실천 항목 요약 노출
- [ ] 타입/빈도 요약 표시
- [ ] 타입 편집 모달 진입/저장 정상
- [ ] 최소/도전 모드 안내 문구 확인
- [ ] 저장 완료 후 세션 종료

---
