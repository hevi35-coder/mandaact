# Mandalart Manual Input AI Suggestion (Draft)

**시작일**: 2026-01-17  
**상태**: 기획 중  
**목적**: 사용자가 직접 입력하는 만다라트 생성 과정에서 AI(LLM)를 활용해 세부목표와 실천항목을 제안함으로써 입력을 돕는다.

---

## 🎯 목표
- **완전 수동 → 하이브리드 자동화**: 모든 칸을 직접 채우는 부담을 줄인다.
- **컨텍스트 기반 제안**: 사용자가 입력한 핵심 목표나 세부 목표에 맞춰 관련성 높은 항목을 실천 가능하도록 제안한다.
- **UX 중심 설계**: 입력을 멈추지 않고도 자연스럽게 제안을 수락하거나 수정할 수 있는 UI 구현.

---

## 🛠 주요 기능 (예정)
1. **세부목표 추천**: 핵심 목표 입력 시 연관된 8개 세부 목표 제안.
2. **실천항목 추천**: 특정 세부 목표에 대해 빈 칸을 채울 수 있는 루틴/미션 제안.
3. **자동 완성**: 입력 중인 텍스트를 바탕으로 나머지 칸을 채워주는 기능.
4. **API 통합**: 기존 `ai-coaching` 인프라를 활용하거나 더 가벼운 전용 Edge Function 구축.
